Ciriú - Ecommerce

  • Angular

  • TypeScript

  • Java

  • Postgresql

  • Docker

  • Bootstrap

  • Mercado pago

Desarrollado a medida para la empresa Ciriú ubicada en la localidad de Hernando, Córdoba, Argentina. Brindandole la solución en la gestión de sus ventas y manipulación de los artículos en su catálogo. El desarrollo se hizo en Java con el framework Spring Boot para el BackEnd, implementando a Mercado Pago como pasarela de pagos, por otro lado, se utilizo Angular para el FrontEnd de la página, y como gestor de base datos se implemento PostgreSQL para almacener la información.
